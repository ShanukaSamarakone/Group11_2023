Addition

Addition is combining two or more numbers to find their total.

Example: If you have 3 apples and you get 2 more apples, how many apples do you have in total?

Explanation:
- You start with 3 apples.
- Then you get 2 more apples.
- Now count all the apples: 1, 2, 3, 4, 5.
- So, 3 apples + 2 apples = 5 apples.
- Therefore, you have 5 apples in total.

Subtraction

Subtraction is taking away one number from another to find the difference.

Example: If you have 5 apples and you give away 2 apples, how many apples do you have in left?

Explanation:
- You start with 5 apples.
- Then you give away 2 apples.
- Now count how many apples are left: 1, 2, 3.
- So, 5 apples - 2 apples = 3 apples.
- Therefore, you have 3 apples left.
